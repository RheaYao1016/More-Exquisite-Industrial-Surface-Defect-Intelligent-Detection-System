# Admin Guide

## Admin Responsibilities
- Configure AI providers
- Manage data source definitions
- Review audit logs
- Oversee training and report workflows

## Startup
- Use `start-platform-lite.bat` for local integrated startup
- Use `start-platform-enterprise.bat` when `DATABASE_URL` is configured

## Recommended Acceptance Checks
- Login and role routing
- AI provider test call
- Data source upload
- Analysis job creation
- Word / Excel / CSV export
- Training job creation
