# Operator Guide

## Recommended Daily Workflow
1. Log in with the `operator` role.
2. Use AI Assistant for shift-level diagnostics.
3. Open Data Hub to inspect newly uploaded files.
4. Use Report Center to download plain-language reports.

## Good Practices
- Prefer concise analysis mode for fast shift review.
- Upload files with clear names and stable schema mapping.
- Re-run reports after updating data sources.
